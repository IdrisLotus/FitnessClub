# Веб-приложение для учета клиентов фитнес-клуба

Дипломный проект на Django.

## Автор

Темаев Идрис Исламович

## Описание проекта

Веб-приложение предназначено для учета клиентов фитнес-клуба.  
Система позволяет работать с клиентами, тренерами, абонементами, посещениями, оплатами, расписанием занятий и отчетами.

## Используемые технологии

- Python
- Django
- SQLite
- Django Templates
- Bootstrap
- Git / GitHub

## Основные модули

- accounts — авторизация и роли пользователей
- clients — учет клиентов
- trainers — учет тренеров
- memberships — абонементы
- visits — журнал посещений
- payments — оплаты
- schedule — расписание занятий
- reports — отчеты
